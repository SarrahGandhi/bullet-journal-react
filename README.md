# Bullet Journal

A beautiful, modern bullet journal web app built with React.

## Features

- **Year at a Glance** - View all 12 months in a calendar grid, see mood colors and note indicators
- **Mood Tracker** - Track your daily mood with 5 options (Amazing, Good, Okay, Meh, Bad)
- **Daily Notes** - Write gratitude entries, daily goals, and detailed notes for each day
- **Theme Customization** - Choose from 6 beautiful themes:
  - Lavender (default)
  - Sage
  - Rose
  - Ocean
  - Sunset
  - Midnight (dark mode)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Data Storage

All journal data is saved locally in your browser using localStorage, so your entries persist between sessions.

## Tech Stack

- React 18
- Vite
- date-fns
- Lucide React Icons
